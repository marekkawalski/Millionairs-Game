   <h2 id="howToPlay">How to play</h2>
  <ol>
  <li>
   Start the program.
  </li>
   <li>
 Go through intro section
  </li>
  <li>
  Choose play
  </li>
  <li>
    Press one of below:
    <ul>
      <li>a, b, c or d to give the answer</li>
      <li><b>@</b> to ask audience for help</li>
      <li><strong>#</strong> to call a friend</li>
      <li><strong>$</strong> to choose 50/50</li>
    </ul>
  </li>
  </ol>
